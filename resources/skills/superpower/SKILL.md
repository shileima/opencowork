# Superpower（默认技能）

本技能随客户端安装时默认安装，用于提供开箱即用的增强能力与规范。

## 何时使用

- 用户安装/首次使用客户端后，本技能已自动安装到本地技能目录，无需额外操作。
- 可在本 SKILL.md 中补充你的团队或个人的常用规范、快捷指令、模板说明等，Agent 会据此提供更一致的回答。

## 扩展方式

在 `SKILL.md` 中增加章节即可，例如：

- **常用命令**：团队约定的脚本、工具调用方式
- **命名与格式**：项目命名、分支、提交信息规范
- **协作约定**：与其它技能（如 ai-playwright、generate-file）的配合方式

保存后 Agent 会在后续对话中自动加载更新后的内容。
