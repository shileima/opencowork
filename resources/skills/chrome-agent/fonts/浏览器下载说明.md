# 浏览器下载字体（推荐方法）

由于命令行工具可能无法使用 VPN，请使用浏览器下载：

## 🚀 快速步骤

### 1. 打开浏览器（已连接 VPN）

访问以下链接：
```
https://github.com/notofonts/noto-cjk/releases
```

### 2. 找到正确的文件

- 查找 **"Sans"** 版本的 releases（不是 Serif）
- 在 Assets 列表中找到 **"Subset TTF"** 或 **"Variable TTF"** 部分
- 下载：`NotoSansCJK-Regular.ttf` 或 `NotoSansCJK-SC-Regular.ttf`
- ⚠️ **重要**：必须下载 **TTF** 格式，不要下载 TTC

### 3. 放置文件

将下载的 `.ttf` 文件放到：
```
~/.qa-cowork/skills/chrome-agent/fonts/
```

### 4. 重命名（如果需要）

如果文件名不是 `NotoSansCJK-SC-Regular.ttf`，请重命名为：
```
NotoSansCJK-SC-Regular.ttf
```
或
```
NotoSansCJK-Regular.ttf
```

### 5. 验证

运行测试脚本：
```bash
cd ~/.qa-cowork/skills/chrome-agent
node test-pdf-chinese.js
```

如果看到：
```
✅ 已加载中文字体: /path/to/NotoSansCJK-SC-Regular.ttf
```

说明成功！

## 📝 直接下载链接

如果 releases 页面加载缓慢，可以尝试直接下载：

**简体中文版本**：
```
https://github.com/notofonts/noto-cjk/releases/download/Sans2.004/07_NotoSansCJK-Regular.ttc.zip
```

⚠️ **注意**：这个链接下载的是 TTC 格式，需要解压后提取 TTF。

**更好的选择**：在 releases 页面查找 "Subset TTF" 部分的单独 TTF 文件。

## ✅ 完成后的效果

下载并放置字体文件后，运行 `weibo-scraper.js` 脚本生成的 PDF 将正确显示中文，不再出现乱码。
